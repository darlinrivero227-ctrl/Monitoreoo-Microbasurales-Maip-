# **App Name**: Maipú Verde

## Core Features:

- Current Air Quality Dashboard: Displays real-time air quality metrics like the '612' gas presence index and health status alerts in a high-visibility hero card.
- Interactive Donut Weekly Summary: Visual representation of the week's air conditions categorized by Clean, Moderate, Deteriorated, and High Presence of Gases.
- Citizen Contribution Tool: A portal for residents to submit local environmental reports and data through the 'Aporte Ciudadano' feature.
- AI Health Advisor Tool: A generative AI tool that analyzes current levels and uses reasoning to provide personalized outdoor activity recommendations and precautions.
- Historical Peak & Average Trackers: Dynamic UI cards displaying calculated weekly averages and historical peaks like the current 672 index mark.

## Style Guidelines:

- Primary Color: Vibrant Forest Green (#27A670) reflecting environmental care and freshness.
- Background Color: Mint White (#F5F9F7), providing a clean, airy, and professional backdrop for data cards.
- Accent Color: Warning Crimson (#E53E3E) used specifically for critical alerts like 'Alta presencia de gases'.
- Font pairing: 'Space Grotesk' for tech-oriented numerical data and headlines to ensure high readability of indices, paired with 'Inter' for clear, objective body text.
- Outlined alert icons and weather-specific symbols using a uniform stroke width to match the modern card aesthetic.
- A strictly vertical single-column mobile-first layout with generous white space and soft shadow-bordered cards for depth.
- Gentle fade-in transitions for cards as the user scrolls, and smooth radial drawing animations for the weekly donut chart.
